# 合戦ズ プロトタイプ（縦切り試作）

長久手市文化の家『合戦ズ』(作: 麻原奈未) を原作にした短編ストーリーRPGの **縦切り試作** です。
`移動 → 会話 → 1戦闘` の最小ループを確認するためのもので、**このコードがそのまま本編エンジンの土台**になります。

- 技術: Vanilla JS（HTML5 Canvas）。**依存ゼロ・ビルド不要**。
- 完全無料。GitHub Pages 等にそのまま載せて一般公開できます。
- PC・スマホ（ブラウザ）対応。

## 遊び方

### いちばん簡単（PC）
`index.html` を**ダブルクリック**でブラウザが開きます。

### おすすめ（ローカルサーバ・スマホ確認も可）
このフォルダで以下を実行し、表示されたURLを開きます。
```
python3 -m http.server 8000
```
→ ブラウザで http://localhost:8000/ を開く（同じWi-Fiならスマホからも `http://（PCのIP）:8000/` で確認可）。

| | キーボード | スマホ |

## 今回入っているもの（縦切りの中身）
1. **タイトル** → はじめる
2. **記念館（現代）**：オダ（女性・短髪）を操作し、屏風・甲冑・刀を調べる（原作のギャグ）→ 出口でタイムスリップ
3. **戦国の長久手（古戦場）**：いけ（池田輝政）・みち（林通具）と会話、勝入塚を調べる（史実）
4. **戦闘**：`たたかう / みやぶる / にげる` のコマンドバトル。
   - 初戦は「みやぶる」で弱点を見抜くまで攻撃が通らない（観察→攻撃を教える設計）
   - たたかうは**たまに急所/空振り**、敵の攻撃も**たまに回避**
   - 「みやぶる」はオダのレベルで強化（観察→看破→心眼。上位は**確率発動**）
   - **ダメージ表示・ヒット閃光・画面揺れ・画面端の陰影（ビネット）**の演出
5. **レベル制**：勝利で**経験値→レベルアップ**（最大HP・こうげき・みやぶるが成長）
6. **ランダムエンカウント**：初戦後、草地を歩くと たまに もののけ が出現（**フラッシュ演出付き**／レベル上げ）
7. **本編シナリオ＝原作『合戦ズ』を 開幕〜〈幕〉まで 通しで再現**：
   - **序章（記念館）**：踊り子の舞 → オダの ご案内練習（秀吉/家康/家康派）→ 屏風・甲冑・刀 → 出口で **いけ登場のフルコメディ**（覚悟→刀なし／徳川／切腹／記念写真＝首取り／めがね＝南蛮渡来／武器蔵→種子島＝硫黄＝ロケット／池田輝政の名乗り＝大包平・刀剣乱舞）
   - **一章**：**みち登場と再会**（てる！人間無骨！長可を探して／固い抱擁）→ 修行（チュートリアル戦）
   - **二章**：戦の解説（羽柴＝柴漬けネタ／秀吉vs家康 唯一の直接対決／中入り作戦 → 井伊直政の赤備え初陣／森長可vs井伊の“ビジュ対決”／池田恒興＝信長の乳兄弟・冒険好き → 挟み撃ち・森長可/恒興の討死・勝入塚）
   - **三〜五章**：みちの覚悟（刀を向ける・父の仇・生き恥）→ **天下論**（天下とは／信長の真意＝誰も殺し合わぬ世／来世を見たい）→ **現代の長久手**（空飛ぶ乗り物／勝入塚に花）→ **別れ**（心して、生きろ！／達者でな）→ 踊り子＝落武者・一国に替え難い名刀
   - **エピローグ（館長）**：閏月のお化け／四月九日＝天正の戦の日／武将が集う／オダ＝信長の末裔疑惑／砂子の交差点の一蘭／家康っぽい → 踊り子の舞で **〈幕〉**
   - **歴史考証**：いけ・みちは **オダを未来人だと思っていない**（オダの未来知識は内心のひとりごと）。「小牧・長久手の戦い」「古戦場」は後世呼称で 武将は使わない／秀吉＝当時「羽柴」／大包平は「のちに授かる」。**行頭禁則**・エンカウント控えめ・場面転換フラッシュ。
8. **ステータスメニュー**（フィールドで X / B キャンセル）：**ステータス／そうび／史跡図鑑／武将名鑑**の4タブ。装備の付け替え（武器・防具で こうげき/まもりが変化）、図鑑・名鑑は 発見した史跡・武将のみ閲覧可（未発見は「？？？」）。
9. **セーブ／つづきから**：戦国フィールドの **篝火（かがりび）** を調べると localStorage に保存。タイトルに「つづきから」が出て、続きから再開できる。
10. **史跡めぐり（サイドクエスト・学習モード）**：タイトルの「史跡めぐり」から。長久手の実在史跡 **5か所（色金山／御旗山／血の池公園／武蔵塚／安昌寺＝首塚）** を めぐり、各地で **小エピソードで学ぶ → 歴史クイズに答える → その地の もののけと戦う** の流れ。勝つと **史跡図鑑が解放**＆「踏破ずみ ✓」。クイズ正解で相手が油断（HP減）。**5か所すべて踏破で「赤備えの具足」の写しを入手**。シティプロモーション＆歴史学習の中心機能。
